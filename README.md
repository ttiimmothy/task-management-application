# Task Manager
[![CI](https://github.com/ttiimmothy/task-manager/actions/workflows/ci.yaml/badge.svg)](https://github.com/ttiimmothy/task-manager/actions/workflows/ci.yaml)

A task manager using `Typescript`, `React.js` amd `Vite`. It uses `React bootstrap` and `tailwindCSS` for simplifying CSS usage. It uses `Formik` and `Yup` to make the create task form and uses `react-dnd` so the tasks can be dragged and dropped to different **status**. The manager uses `react-redux` and `@redux/toolkit` for state management.
